# Wiki Scope

## Topic Areas

### Foundations
LLM architecture, transformer mechanism, training pipeline (pre-training, instruction tuning, RLHF, DPO), tokenisation, context windows, statelessness, non-determinism, post-training alignment, multimodal capabilities.

### Agent Architectures
Single-agent systems (ReAct, tool calling, MCP), multi-agent taxonomies (10-pattern classification), coordination mechanisms (orchestrator-worker, debate, shared room), memory systems (episodic, semantic, procedural), agent frameworks (AutoGen, CrewAI, MetaGPT, LangGraph).

### Failure Modes
Hallucination (types, causes, detection), calibration and confidence, sycophancy, context window degradation (lost-in-the-middle), tool-calling unreliability, prompt sensitivity, model collapse, multi-agent failure modes (flooding, context divergence, cascading errors).

### Safety and Reliability
Risk properties of stochastic systems, epistemic independence, common-cause failure in AI, monoculture collapse, defence-in-depth at the reasoning layer, V&V challenges, non-deterministic system testing, regulatory frameworks for AI.

### Human-AI Interaction
Automation bias, trust calibration (over-trust, under-trust), situation awareness in human-AI teams, team cognition, skill degradation, cognitive load, authority and decision-making, operator override behaviour.

### Design Patterns
Human-system interface design for AI advisors, delivery modes (flow gates, room pause), governance gates, knowledge grounding (RAG, knowledge graphs), guardrails, transparency and explainability, alert design.

### Evaluation and Testing
Benchmarking LLMs for domain tasks, prototyping methodologies, HRA methods for AI-assisted operations (SPAR-H, IDHEAS-ECA, ATHEANA), PSA integration, evaluation frameworks, capability gradient approach.

### Domain Applications
Cross-domain scenarios illustrating architectural concepts in nuclear, aviation, oil and gas, and medical settings. Shift handover, anomaly detection, emergency response, multi-unit monitoring, degraded system operation.
